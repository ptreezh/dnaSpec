# DSGS模块成熟化核验器

## 功能概述
dnaspec-modulizer是DSGS系统的模块化技能，专门负责：
1. 自底向上检查系统各模块的成熟度
2. 验证模块是否达到封装标准
3. 对成熟模块进行模块化封装
4. 管理模块化后的组件生命周期
5. 提供模块重构和版本管理功能

## 使用方法
作为Claude Skills的一部分，该技能在系统开发过程中自动调用，或由开发者主动请求模块化检查。

## 核心原则
- **自底向上**: 从最底层组件开始，逐层向上进行模块化
- **成熟度验证**: 只有通过完整测试的模块才能被封装
- **不可变性**: 封装后的模块默认不可修改，确保稳定性
- **显式重构**: 需要重构时必须显式请求

## 输入要求
- 系统架构信息
- 组件测试报告
- 依赖关系图
- 模块化策略

## 输出格式
技能输出详细的模块化检查报告，包括：
- 模块成熟度评估
- 可模块化组件列表
- 封装建议和策略
- 重构风险分析
