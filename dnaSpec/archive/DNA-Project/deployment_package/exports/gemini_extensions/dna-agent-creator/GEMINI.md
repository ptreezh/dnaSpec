# dnaspec-agent-creator 扩展

## 功能描述
DNASPEC智能体创建器子技能，用于根据项目需求创建和配置智能体、定义智能体角色和行为、生成智能体规范文档。当用户提到创建智能体、配置agent、多智能体系统、智能体角色定义、模块智能化、agentic或具身认知时使用此技能。不要在讨论一般系统架构时使用。
