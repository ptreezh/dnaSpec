# Навыки контекстной инженерии DSGS - Краткое руководство (Russian)

## Обзор проекта

Набор навыков контекстной инженерии DSGS (Dynamic Specification Growth System) - это профессиональный инструментарий разработки с поддержкой ИИ, специально разработанный для платформ ИИ командной строки, предоставляющий возможности анализа контекста, оптимизации и когнитивных шаблонов с рабочим процессом безопасности ИИ.

## Основные улучшения

### 1. Унифицированная архитектура навыков
- **Консолидированная реализация**: Стандартный и расширенный режимы объединены в одну реализацию
- **Переключение режимов**: Используйте параметр `mode` для управления уровнем функциональности ('standard' или 'enhanced')
- **Единый интерфейс**: Упрощенный интерфейс, избегающий дублирующей функциональности

### 2. Плоская структура каталогов
- **Один навык на каталог**: Каждый навык в своем собственном каталоге, без ненужной вложенности
- **Упрощенная организация**: Интуитивное расположение навыков, более простое обслуживание
- **Уменьшено запутывание**: Четкие границы навыков, отсутствие перекрывающейся функциональности

### 3. Рабочий процесс безопасности ИИ
- **Временное рабочее пространство**: Сгенерированный ИИ контент сначала сохраняется во временную область
- **Автоуправление**: Автоматическое оповещение при превышении 20 файлов
- **Механизм подтверждения**: Содержимое должно быть проверено перед включением в основной проект
- **Автоматическая очистка**: Автоматическая очистка временного рабочего пространства после завершения задачи

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/ptreezh/dnaSpec.git
cd dsgs-context-engineering

# Установить
pip install -e .
```

## Использование

### Команды CLI
```
/speckit.dsgs.context-analysis "Проанализировать качество этого документа требований" mode=enhanced
/speckit.dsgs.cognitive-template "Как улучшить производительность" template=verification
/speckit.dsgs.context-optimization "Оптимизировать это требование" optimization_goals=clarity,relevance
/speckit.dsgs.architect "Спроектировать архитектуру электронной коммерции"
/speckit.dsgs.git-skill operation=status
/speckit.dsgs.temp-workspace operation=create-workspace
```

### Python API
```python
from clean_skills.context_analysis import execute as context_analysis_execute

# Стандартный режим
result = context_analysis_execute({
    'context': 'Разработать функцию входа пользователя',
    'mode': 'standard'
})

# Расширенный режим
result = context_analysis_execute({
    'context': 'Разработать функцию безопасного входа пользователя',
    'mode': 'enhanced'
})
```

## Рекомендации по безопасности ИИ

1. **Перед генерацией ИИ**: Всегда создавайте временное рабочее пространство
2. **Проверить содержание**: Используйте механизм подтверждения для проверки сгенерированного ИИ контента
3. **Регулярная очистка**: Мониторьте количество временных файлов
4. **Очистить рабочую область**: Очистите временную область после завершения задачи
