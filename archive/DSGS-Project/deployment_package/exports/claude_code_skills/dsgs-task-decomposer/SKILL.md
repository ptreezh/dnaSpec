# DSGS任务分解器子技能

## 功能概述
dnaspec-task-decomposer是DSGS智能架构师系统的子技能，专门负责：
1. 分析复杂项目需求并分解为原子化任务
2. 生成任务依赖关系图和执行顺序
3. 确保任务上下文文档的闭包性
4. 为每个任务生成详细的执行指南
5. 提供任务优先级和资源分配建议

## 使用方法
作为Claude Skills的一部分，该技能会根据主技能的路由被自动调用。

## 输入要求
- 复杂项目需求描述
- 项目约束条件
- 可用资源信息
- 时间限制要求

## 输出格式
技能输出完整的任务分解文档，包括：
- 原子化任务列表
- 任务依赖关系图
- 任务执行顺序
- 任务上下文文档
- 资源分配建议
- 时间估算
