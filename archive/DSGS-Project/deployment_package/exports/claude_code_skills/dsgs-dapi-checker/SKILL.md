# DSGS分布式接口文档检查器 (DAPIcheck)

## 功能概述
dnaspec-dapi-checker是DSGS智能架构师系统的子技能，专门负责：
1. 扫描系统各个组件之间的接口定义
2. 检查接口的全面性和一致性
3. 验证实现与接口文档的一致性
4. 生成接口一致性报告
5. 检测接口依赖和兼容性问题

## 使用方法
作为Claude Skills的一部分，该技能会根据主技能的路由被自动调用。

## 输入要求
- 系统代码库路径
- 接口文档路径
- 接口定义文件（API spec, IDL, etc.）
- 系统架构图或组件关系图

## 输出格式
技能输出完整的接口一致性检查报告，包括：
- 接口实现与文档的差异
- 缺失的接口实现
- 不一致的接口参数
- 潜在的兼容性问题
- 修复建议

## 检查范围
- 外部接口：组件对外暴露的API
- 内部抽象接口：组件间的抽象接口定义
- 数据接口：数据传输格式和协议
- 通信接口：通信协议和消息格式
