# DNASPEC Context Optimization - 上下文优化

## 核心概念

**优化上下文，提升质量和效率**

dnaspec-context-optimization 根据context-analysis的诊断结果，优化上下文内容，提升清晰度、相关性、完整性、一致性和效率。

**与context-analysis的配合**：

```
context-analysis (诊断) → context-optimization (治疗) → context-analysis (复查)
```

**优化策略**：

1. **压缩冗余** - 删除重复内容，保留核心信息
2. **结构化** - 改进组织结构，提升可读性
3. **术语统一** - 统一术语使用，提升一致性
4. **简化表达** - 简化复杂表述，提升清晰度
5. **补充缺口** - 补充缺失信息，提升完整性
