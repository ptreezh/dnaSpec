# Cognitive Template Skill - Brief Documentation

## Overview
The Cognitive Template Skill applies structured thinking frameworks to contextual information processing. It provides predefined templates for common cognitive tasks like chain-of-thought reasoning, few-shot learning, and verification checking.

## Purpose
**Primary Purpose**: Apply structured cognitive frameworks to information processing tasks

## Key Features
- **Multiple Templates**: Chain-of-thought, few-shot learning, verification, role-playing, understanding frameworks
- **Enhanced Modes**: Additional templates for complex problem-solving and decision-making
- **Structured Approaches**: Systematic methods for tackling different types of tasks
- **Reusable Frameworks**: Consistent patterns for similar problems

## Applications
- Problem-solving assistance
- Decision support
- Learning and training
- Quality assurance and verification

## Integration
Works with both Context Analysis and Context Optimization Skills to provide structured approaches to context evaluation and improvement. Used by other skills to apply appropriate thinking frameworks.
