# DNASPEC Cognitive Template - 认知模板

## 核心概念

**可复用的认知模式，提升处理效率和质量**

认知模板是预定义的认知处理模式，帮助快速处理特定类型的任务。模板封装了最佳实践和成功模式。

**模板价值**：

1. **效率提升** - 快速启动，减少思考时间
2. **质量保证** - 基于最佳实践，确保质量
3. **一致性** - 统一处理方式，提升一致性
4. **可复用** - 一次创建，多次使用

**模板类型**：

- **理解模板** - 快速理解问题和需求
- **推理模板** - 系统化推理和分析
- **验证模板** - 全面验证结果质量

**核心原则**：

**渐进式复杂度** - 从简单模板开始，逐步演化到复杂模板
**格式塔整体性** - 模板作为整体，大于各部分之和
**上下文适配** - 根据具体场景调整模板
