# DNASPEC Cognitive Template - 高级应用

## 元模板 (Meta-Template)

### 模板创建模板

```yaml
template_creation_template:
  steps:
    1. 识别模式:
       - 观察成功案例
       - 提取共同模式
       - 定义关键步骤

    2. 设计模板:
       - 定义结构
       - 设计指导
       - 添加示例

    3. 验证模板:
       - 应用测试
       - 收集反馈
       - 迭代改进

    4. 文档化:
       - 编写说明
       - 提供示例
       - 建立索引

  example:
    创建: "代码审查模板"

    识别模式:
      - 成功审查都有明确检查点
      - 都有标准流程
      - 都有反馈机制

    设计模板:
      结构: 准备 → 检查 → 反馈
      指导: 每步详细说明
      示例: 提供审查清单

    验证:
      - 团队试用
      - 收集反馈
      - 改进流程

    文档化:
      - 编写使用指南
      - 提供示例审查
      - 建立模板库
```
