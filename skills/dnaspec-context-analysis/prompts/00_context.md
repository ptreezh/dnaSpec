# DNASPEC Context Analysis - 上下文质量分析

## 核心概念

**分析上下文质量，确保系统演化中的质量一致性**

dnaspec-context-analysis 分析上下文在5个维度的质量：清晰度、相关性、完整性、一致性、效率。

**五大质量维度**：

1. **清晰度 (Clarity)** - 内容是否易于理解？
2. **相关性 (Relevance)** - 内容是否与目标相关？
3. **完整性 (Completeness)** - 信息是否完整？
4. **一致性 (Consistency)** - 术语、逻辑是否一致？
5. **效率 (Efficiency)** - 表达是否简洁高效？

**主要功能**：

- 多维度质量评估（5个维度，每个0-1评分）
- 演化质量监控（确保从简单到复杂的质量一致性）
- 风险检测（上下文爆炸/腐化风险）
- 缺口识别（信息缺口、不一致点）

**核心价值**：
- 诊断上下文质量问题
- 识别改进空间
- 与context-optimization形成改进循环
- 防止质量随复杂度下降
