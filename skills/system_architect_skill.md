# System Architect Skill - Brief Documentation

## Overview
The System Architect Skill designs complete software architectures based on requirements and constraints. It generates comprehensive architectural blueprints covering components, data flows, technology stacks, and deployment strategies.

## Purpose
**Primary Purpose**: Design complete software system architectures from requirements

## Key Features
- **Comprehensive Architecture Design**: Components, data flows, technology recommendations
- **Multi-Layered Approach**: Presentation, business logic, data, and infrastructure layers
- **Technology Stack Guidance**: Framework and tool recommendations
- **Quality Attribute Consideration**: Security, performance, scalability, maintainability

## Applications
- Initial system design and planning
- Architecture review and improvement
- Technology selection and evaluation
- Deployment strategy planning

## Integration
Provides architectural foundations that guide Modularization Skill decomposition efforts. Works with DAPI Check Skill to ensure interface compliance with architectural designs.
