# DNASPEC Git - 版本管理辅助

## 核心概念

**智能Git辅助，而非强制执行**

dnaspec-git 是一个辅助工具，帮助你更有效地使用Git进行版本管理。它不强制规则，而是提供建议和辅助。

**三大核心原则**：

1. **建议而非强制** - 提供建议，由用户决策
2. **智能辅助** - 分析变更，生成规范的提交信息
3. **上下文感知** - 根据项目特点定制建议

**主要功能**：

- 智能提交信息生成（Conventional Commits）
- 分支管理辅助（命名规范、策略推荐）
- 合并冲突解决指导（冲突分析、解决建议）
- 版本标签管理（语义化版本）
- Git工作流优化（根据项目特点选择工作流）

**核心理念**：整体性的Git工作流，理解变更意图和影响，提供连贯的版本管理过程。
