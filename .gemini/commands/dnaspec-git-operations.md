## Usage
Use this command to establish Git constitution and rules that prevent AI-generated file pollution in projects.

```bash

/dnaspec.git-operations "operation=setup-constitution project_path=."

# Install Git hooks for AI file protection
/dnaspec.git-operations "operation=install-hooks project_path=."

# Smart commit with DNASPEC rules
/dnaspec.git-operations "operation=smart-commit project_path=. message='[DNASPEC] Add AI-validated feature'"

# Clean workspace from AI temporary files
/dnaspec.git-operations "operation=clean-workspace project_path=."

# Enforce Git rules
/dnaspec.git-operations "operation=enforce-rules project_path=."

# Get workspace status
/dnaspec.git-operations "operation=status-report project_path=."
```
