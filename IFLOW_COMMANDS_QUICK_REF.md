## 📋 8 个 DNASPEC 命令

### 1️⃣ 系统架构师
```
/dnaspec.architect 设计[微服务/单体/无服务器]架构
```

### 2️⃣ 任务分解器
```
/dnaspec.task-decomposer 将[需求/功能]分解为开发任务
```

### 3️⃣ 代理创建器
```
/dnaspec.agent-creator 创建[代码审查/测试/部署]自动化代理
```

### 4️⃣ 约束生成器
```
/dnaspec.constraint-generator 生成[安全/验证/业务]约束
```

### 5️⃣ DAPI 检查器
```
/dnaspec.dapi-checker 验证[API/接口]设计
```

### 6️⃣ 模块化器
```
/dnaspec.modulizer 从代码中提取可重用模块
```

### 7️⃣ 缓存管理器
```
/dnaspec.cache-manager 设计缓存和性能优化策略
```

### 8️⃣ Git 操作
```
/dnaspec.git-operations 清理和优化 Git 仓库
```

## 💡 快速示例

### 场景：开发新功能

```
# 1. 设计架构
/dnaspec.architect 设计用户认证系统的微服务架构

```
