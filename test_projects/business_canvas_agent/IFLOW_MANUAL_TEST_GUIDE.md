## 🔧 遇到问题？

### iflow 并发限制
```
您当前的账号已达到平台并发限制
```

**解决**:
1. 关闭所有其他 iflow 实例
2. 等待 1-2 分钟
3. 重启 iflow

### 命令不识别
```
未知命令：/speckit.dnaspec.architect
```

**解决**:
1. 检查 Stigmergy 钩子是否存在
2. 重新部署: `dnaspec deploy --force-stigmergy`
3. 重启 iflow

### 输出质量问题

**解决**:
1. 提供更详细的输入
2. 明确需求和期望
3. 使用结构化提示
