# 用户需求 - 智选电商平台

## Epic 1: 用户认证与授权

### US-1.1 用户注册
**作为** 新用户
**我想要** 注册账号
**以便** 我可以购买商品

**验收标准**:
- 支持邮箱/手机号注册
- 验证码验证
- 密码强度要求：至少8位，包含字母和数字
- 注册成功后自动登录

### US-1.2 用户登录
**作为** 已注册用户
**我想要** 登录账号
**以便** 我可以访问购物车和订单

**验收标准**:
- 支持邮箱/手机号+密码登录
- 记住我功能（7天免登录）
- 密码错误3次锁定账号30分钟
- 支持找回密码

## Epic 2: 商品浏览与搜索

### US-2.1 商品列表浏览
**作为** 用户
**我想要** 浏览商品列表
**以便** 我可以找到想要的商品

**验收标准**:
- 支持分页（每页20个商品）
- 支持按价格、销量、评分排序
- 支持筛选（分类、品牌、价格区间）
- 显示商品缩略图、标题、价格

### US-2.2 商品搜索
**作为** 用户
**我想要** 搜索商品
**以便** 快速找到目标商品

**验收标准**:
- 支持关键词搜索
- 搜索结果高亮显示
- 搜索历史记录
- 热门搜索推荐

### US-2.3 商品详情
**作为** 用户
**我想要** 查看商品详情
**以便** 了解商品信息

**验收标准**:
- 商品图片轮播
- 规格选择（颜色、尺寸等）
- 库存状态显示
- 用户评价展示
- 相关商品推荐

## Epic 3: 购物车与下单

### US-3.1 添加购物车
**作为** 用户
**我想要** 将商品加入购物车
**以便** 稍后购买

**验收标准**:
- 支持添加多个规格
- 实时库存校验
- 购物车数量显示
- 本地缓存（未登录也可添加）

### US-3.2 购物车管理
**作为** 用户
**我想要** 管理购物车
**以便** 调整购买数量

**验收标准**:
- 修改商品数量
- 删除商品
- 批量操作
- 价格自动计算

### US-3.3 提交订单
**作为** 用户
**我想要** 提交订单
**以便** 完成购买

**验收标准**:
- 收货地址管理
- 配送方式选择
- 支付方式选择
- 订单确认页
- 库存预占（15分钟）

## Epic 4: 订单管理

### US-4.1 订单列表
**作为** 用户
**我想要** 查看我的订单
**以便** 了解订单状态

**验收标准**:
- 订单状态筛选
- 订单详情查看
- 物流信息追踪
- 订单操作（取消、申请退款）

### US-4.2 订单支付
**作为** 用户
**我想要** 支付订单
**以便** 完成交易

**验收标准**:
- 支持微信支付
- 支持支付宝
- 支持信用卡
- 支付超时自动关闭（30分钟）

## Epic 5: 商家后台

### US-5.1 商品管理
**作为** 商家
**我想要** 管理商品
**以便** 上架销售

**验收标准**:
- 商品上架/下架
- 库存管理
- 价格设置
- 批量导入

### US-5.2 订单处理
**作为** 商家
**我想要** 处理订单
**以便** 发货

**验收标准**:
- 订单列表（按状态筛选）
- 批量发货
- 物流单号录入
- 订单备注

### US-5.3 数据统计
**作为** 商家
**我想要** 查看数据统计
**以便** 了解经营情况

**验收标准**:
- 销售额统计（日/周/月）
- 订单量统计
- 商品销量排行
- 用户增长曲线

## 约束条件

### 安全性
- 用户密码必须加密存储（bcrypt）
- 敏感操作需要二次验证
- API 接口限流（防刷）
- SQL 注入防护

### 性能
- 首页加载时间 < 1秒
- API 响应时间 < 200ms
- 支持并发用户数 > 10000
- 数据库查询优化（索引、缓存）

### 兼容性
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 移动端响应式设计
- 支持 iOS 12+、Android 8+
