# DNASPEC约束生成器子技能

## 功能概述
dnaspec-constraint-generator是DNASPEC智能架构师系统的子技能，专门负责：
1. 分析项目需求并生成相应的系统约束
2. 生成API规范约束和接口定义
3. 定义数据约束和验证规则
4. 创建质量约束和性能指标
5. 生成约束文档和验证机制

## 使用方法
作为Claude Skills的一部分，该技能会根据主技能的路由被自动调用。

## 输入要求
- 项目需求描述
- 架构设计文档
- 技术约束条件
- 质量要求规范

## 输出格式
技能输出完整的约束生成结果，包括：
- 系统约束定义
- API约束规范
- 数据约束规则
- 质量约束标准
- 约束验证机制
