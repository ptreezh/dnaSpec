# DNASPEC智能体创建器子技能

## 功能概述
dnaspec-agent-creator是DNASPEC智能架构师系统的子技能，专门负责：
1. 分析项目需求并创建合适的智能体类型
2. 定义智能体的角色、职责和行为模式
3. 生成智能体规范文档和配置文件
4. 配置智能体间的通信和协作机制
5. 提供智能体监控和管理建议

## 使用方法
作为Claude Skills的一部分，该技能会根据主技能的路由被自动调用。

## 输入要求
- 项目需求描述
- 智能体类型要求
- 协作模式要求
- 性能约束条件

## 输出格式
技能输出完整的智能体创建结果，包括：
- 智能体角色定义
- 智能体行为规范
- 智能体配置文件
- 通信协议定义
- 监控和管理策略
