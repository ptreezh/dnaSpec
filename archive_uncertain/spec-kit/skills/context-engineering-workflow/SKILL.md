# Context Engineering Workflow

## Overview
Comprehensive Context Engineering workflow that combines multiple specialized skills into a complete process for context improvement. This workflow follows the complete Context Engineering methodology from initial analysis through optimization to cognitive enhancement.

The workflow orchestrates the following specialized skills:
- **context-analysis-enhanced**: For detailed context assessment
- **context-optimization-enhanced**: For targeted improvements
- **cognitive-template-enhanced**: For structured thinking application

## Workflow Process

### Phase 1: Context Assessment
Uses the enhanced context analysis capabilities to evaluate the input across multiple dimensions:
- Clarity scoring and issue identification
- Relevance assessment with alignment analysis
- Completeness evaluation with gap identification
- Consistency checking for internal coherence
- Efficiency measurement for information density

### Phase 2: Targeted Optimization
Based on the analysis results, applies optimization techniques to address identified issues:
- Clarity enhancement through language refinement
- Relevance improvement by focusing content
- Completeness enhancement by filling gaps
- Consistency improvement through standardization
- Efficiency optimization by reducing redundancy

### Phase 3: Cognitive Enhancement
Applies appropriate cognitive templates to improve the structure and reasoning:
- Chain of thought for complex problem decomposition
- Few-shot learning for example-based guidance
- Verification framework for quality assurance
- Role playing for perspective enhancement
- Understanding framework for deep comprehension

## Integration Points

### Analysis to Optimization Pipeline
- Analysis results directly inform optimization priorities
- Identified issues become optimization targets
- Metric scores guide the optimization approach
- Specific recommendations are implemented systematically

### Optimization to Cognitive Enhancement
- Optimized content is prepared for cognitive structuring
- Enhancement opportunities are identified
- Appropriate cognitive templates are selected
- Structured thinking patterns are applied
